How to run:
run main.py
  
Controls:  
TAB - switch mode (editor/view)  
  
view mode:  
SPACE - pause/unpause  
'+' or '-' - increase/decrease max fps  
A - Show/Hide ants  
P - Show/Hide pheromones  
G - Show/Hide grid  
  
editor mode (Lclick to place, Rclick to erase):  
O - obstacles  
P - pheromones  
A - ants  
F - food source
